## Как запустить

### Локально
Просто открой `index.html` в браузере.

### GitHub + Vercel
1. Создай новый репозиторий на GitHub
2. Залей все файлы из этой папки (вместе с изображениями в `assets/`)
3. В Vercel: New Project → Import репо → Deploy
4. Готово! Ссылка вида `*.vercel.app`
